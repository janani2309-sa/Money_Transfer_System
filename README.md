# Money Transfer System

An enterprise digital money transfer microservice application built progressively from scratch using Java 17, Spring Boot, MySQL, Angular, and Snowflake.

## Project Structure

This repository is structured as a multi-module progressive system:
* **`backend/`**: Core banking API built with Spring Boot 3.x, Hibernate/JPA, and Spring Security.
* **`frontend/`**: Single Page Application built with Angular 18+ and Angular Material.
* **`database/`**: Relational database schema (`schema.sql`) and data seeding script (`seed-data.sql`) for MySQL.
* **`snowflake/`**: Data analytics queries and warehouse scripts for business intelligence.
* **`docs/`**: Project documentation, diagrams, and developer guides.

## Technologies Used

* **Language**: Java 17 LTS, TypeScript
* **Backend Framework**: Spring Boot 3.x, JPA / Hibernate, Spring Security, Spring AOP
* **Frontend Framework**: Angular 18+, RxJS, Angular Material
* **Database**: MySQL 8.x
* **Analytics**: Snowflake Cloud Data Warehouse
* **Build Tools**: Maven, npm
